## Roles y cuentas de prueba

| Rol         | Permisos principales                           | Formato de email                | Ejemplo                       |
| ----------- | ---------------------------------------------- | ------------------------------- | ----------------------------- |
| **Owner**   | Control total del panel de administración      | `{prefijo}.owner@fitswap.net`   | `omatple.owner@fitswap.net`   |
| **Admin**   | Permisos limitados del panel + vista completa  | `{prefijo}.admin@fitswap.net`   | `omatple.admin@fitswap.net`   |
| **Coach**   | Gestiona sus grupos, añade recetas y alimentos | `{prefijo}.coach@fitswap.net`   | `omatple.coach@fitswap.net`   |
| **Student** | Sin acceso al panel de administración          | `{prefijo}.student@fitswap.net` | `omatple.student@fitswap.net` |
